# 기본 문법

## 자료형
- 기본형: byte, short, int, long, float, double, char, boolean
- 참조형: 클래스, 배열, 인터페이스

## 변수
- 클래스명은 대문자로 시작함
- 변수명은 소문자로 시작함
- 상수는 대문자와 언더스코어 사용함

## 진법 변환
- `Integer.toBinaryString(n)`: 10진수 → 2진수
- `Integer.parseInt(s, 2)`: 2진수 문자열 → 10진수
- 2의 보수: 음수 표현 방식

## 연산자
- 산술: `+`, `-`, `*`, `/`, `%`
- 비교: `==`, `!=`, `>`, `<`, `>=`, `<=`
- 논리: `&&`, `||`, `!`
- 삼항: `조건 ? 참 : 거짓`
- 비트: `&`, `|`, `^`, `~`, `<<`, `>>`

## 조건문
- `if-else`: 조건에 따른 분기
- `switch-case`: 값에 따른 분기
- switch 표현식(Java 12+): `->` 문법 사용
- yield: switch 블록에서 값 반환

## 반복문
- `for`: 횟수가 정해진 반복
- `while`: 조건이 참인 동안 반복
- `do-while`: 최소 1회 실행 후 조건 확인
- `break`: 반복 종료
- `continue`: 현재 반복 건너뜀

## String 비교
- `==`: 참조 비교 (같은 객체인지)
- `equals()`: 값 비교 (내용이 같은지)
- 리터럴 문자열은 String Pool에 저장됨
- `new String()`은 힙에 새 객체 생성함
