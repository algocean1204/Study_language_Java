# 생성자와 메서드 오버로딩

## 생성자
- 객체 생성 시 호출되는 특별한 메서드
- 클래스명과 동일한 이름 사용함
- 반환타입 없음
- 필드 초기화에 사용함

## 기본 생성자
- 매개변수 없는 생성자
- 명시적 생성자가 없으면 컴파일러가 자동 생성함
- 명시적 생성자가 있으면 자동 생성 안됨

## 생성자 오버로딩
- 같은 이름, 다른 매개변수의 생성자 여러 개 정의
- 다양한 방식으로 객체 초기화 가능함

```java
public Car() { }
public Car(String model) { }
public Car(String model, String color) { }
```

## 메서드 오버로딩
- 같은 이름, 다른 매개변수의 메서드 정의
- 매개변수 개수 또는 타입이 달라야 함
- 반환타입만 다른 것은 오버로딩 아님

```java
int add(int a, int b) { }
double add(double a, double b) { }
int add(int a, int b, int c) { }
```

## this 키워드
- 현재 객체 자신을 참조함
- 매개변수와 필드명이 같을 때 구분에 사용함
- `this()`: 같은 클래스의 다른 생성자 호출
