# 데몬 스레드 (Daemon Thread)

## 정의
- 주 스레드의 보조 역할을 하는 스레드
- 주 스레드가 종료되면 자동으로 종료됨
- 백그라운드 작업에 사용함

## 설정 방법
```java
thread.setDaemon(true);  // start() 호출 전에 설정
thread.start();
```

## 특징
- 주 스레드 종료 시 강제 종료됨
- 자동 저장, 가비지 컬렉션 등에 활용됨
- `start()` 호출 전에 `setDaemon(true)` 설정 필수

## 일반 스레드와 차이
| 구분 | 일반 스레드 | 데몬 스레드 |
|------|-----------|-----------|
| 종료 | 작업 완료까지 실행 | 주 스레드 종료 시 강제 종료 |
| 용도 | 주요 작업 | 보조 작업 |
