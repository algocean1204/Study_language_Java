# 상속 (Inheritance)

## 정의
- `extends` 키워드 사용함
- 부모 클래스의 필드와 메서드 물려받음
- 다형성 지원을 위해 만들어짐

## 특징
- 자바는 단일 상속만 지원함 (다중상속 X)
- 모든 클래스는 Object를 상속함
- private 멤버는 상속되나 직접 접근 불가함

## 상속 구현
```java
public class Car { }               // 부모 클래스
public class Truck extends Car { } // 자식 클래스
```

## 생성자 호출 순서
- 자식 생성자의 첫 라인에서 부모 생성자 호출함
- `super()`: 부모 생성자 명시적 호출
- 부모 객체 먼저 생성 후 자식 객체 생성됨

```java
public Truck() {
    super();  // 부모 생성자 호출 (생략 가능)
}
```

## 메서드 오버라이드
- 부모 메서드를 자식에서 재정의함
- 메서드 시그니처가 동일해야 함
- `@Override` 어노테이션 사용 권장함

## super 키워드
- 부모 클래스 참조에 사용함
- `super.필드명`: 부모 필드 접근
- `super.메서드명()`: 부모 메서드 호출
- `super()`: 부모 생성자 호출
